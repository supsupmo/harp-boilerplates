# Harp Boilerplate, Public Style
Harp boilerplate set up with the following:

```
public-style/
  |- harp.json
  +- public/                 <--- public directory
      |- _layout.jade
      |- 404.jade
      |- index.jade
      +- css/
          |- app.scss
      +- images/
      +- js
          |- jquery.min.js
          |- app.js
```
