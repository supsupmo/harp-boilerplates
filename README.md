# Harp Boilerplates
A personal collection of boilerplates for Harp.

## public-style

```
public-style/
  |- harp.json
  |- README.md               <--- won’t be served
  |- secrets.txt             <--- won’t be served
  +- public/                 <--- public directory
      |- _layout.jade
      |- 404.jade
      |- index.jade
      +- css/
          |- app.scss
      +- images/
      +- js
          |- jquery.min.js
          |- app.js
```
